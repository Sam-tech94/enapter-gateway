# Enapter Gateway Landing Page

![Enapter Gateway Landing Page](/enapter_gateway/public/gateway.png)

## Overview

The Enapter Gateway Landing Page is a website designed to showcase the Enapter Gateway Software, a powerful solution for energy management and IoT control. The landing page provides information about the software, its features, and how to download the latest version.

## Features

- Upload the latest firmware version to create a bootable USB flash with Enapter Software for your gateway.
- Download the latest version of the Enapter Gateway Software directly from the landing page.
- Access previous versions of the Enapter Gateway Software for compatibility or reference purposes.
- Connect with the Enapter community and get support on Discord.

## Technologies Used

- **Astro:** Astro is a modern static site generator that allows developers to build websites using JavaScript and HTML. It offers a component-based architecture, allowing for efficient development and easy integration of dynamic functionality.
- **Tailwind CSS:** Tailwind CSS is used for styling the components.
- **Material Icons:** Material Icons are used for graphical elements.

## Getting Started

To run the Enapter Gateway Landing Page locally, follow these steps:

1. Clone the repository to your local machine.
2. Install dependencies using `npm install`.
3. Start the development server with `npm run dev`.
4. Open your web browser and navigate to `http://localhost:3000`.
